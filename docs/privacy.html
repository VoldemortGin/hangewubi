隐私政策

概述

晗戈五笔是一款注重用户隐私的输入法应用。我们不会收集、存储或传输任何用户个人数据。

数据收集

本应用不收集任何个人数据，包括但不限于：

  • 您输入的文字内容
  • 键盘使用习惯或统计信息
  • 设备标识符或位置信息
  • 联系人、照片或其他个人文件

本地处理

所有数据处理均在您的设备本地完成。输入法的全部功能（包括候选词生成、编码匹配等）均在设备端运行，无需网络连接。

网络通信

本应用不会向任何服务器传输数据。应用无需联网即可正常使用。

第三方服务

本应用不包含任何第三方分析工具、广告 SDK 或用户追踪组件。

联系方式

如有任何隐私相关问题，请联系：gin.linhan@gmail.com

English

Privacy Policy

Overview

HangeWubi (晗戈五笔) is an input method application that respects user privacy. We do not collect, store, or transmit any personal data.

Data Collection

This application does not collect any personal data, including but not limited to:

  • Text you type
  • Keyboard usage patterns or statistics
  • Device identifiers or location information
  • Contacts, photos, or other personal files

Local Processing

All data processing happens entirely on your device. All features of the input method, including candidate generation and code matching, run locally without requiring a network connection.

Network Communication

This application does not transmit any data to any servers. The app functions fully offline.

Third-Party Services

This application does not include any third-party analytics, advertising SDKs, or user tracking components.

Contact

For any privacy-related questions, please contact: gin.linhan@gmail.com

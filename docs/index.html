开始使用

从源码构建安装，简单几步即可上手



macOS

可用

通过源码构建，支持 Apple Silicon 和 Intel

# 从源码构建
git clone https://github.com/VoldemortGin/hangewubi.git
cd hangewubi
bash platform/macos/build.sh
cp -r target/macos-app/HangeWubi.app ~/Library/Input\ Methods/
# 注销重新登录，在系统设置 > 键盘 > 输入源中添加
🐧

Linux (IBus)

可用

通过 IBus 框架集成，支持主流 Linux 发行版

# 安装到 IBus
git clone https://github.com/VoldemortGin/hangewubi.git
cd hangewubi
bash platform/linux/install.sh
# 在 IBus 设置中添加晗戈五笔
💻

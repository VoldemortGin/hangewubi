# 晗戈五笔 (HangeWubi) — 产品需求文档

## 1. 产品概述

**晗戈五笔**是一款基于 Rust 开发的五笔输入法引擎，参考清歌五笔的轻量纯粹理念和搜狗五笔的功能全面性，打造一个高性能、可定制、开源的五笔输入法。

**项目名称**：晗戈五笔 (HangeWubi)
**开发语言**：Rust
**目标平台**：先做跨平台引擎核心（CLI 可测试），后续集成 macOS InputMethodKit
**编码方案**：86版五笔（首选），预留98版/新世纪版扩展接口

## 2. 核心需求

### 2.1 码表引擎 (P0)

- 加载 86版五笔码表（TSV 格式：`编码\t汉字\t权重`）
- 支持单字和词组混合码表
- 精确匹配查询（HashMap，O(1)）
- 前缀匹配查询（Trie，用于候选补全）
- 码表热加载（修改后可重新加载，无需重启）

### 2.2 输入引擎 (P0)

- 编码缓冲区管理（最长4码）
- 候选词生成与排序（按权重排序）
- 4码唯一自动上屏
- 空格键选择首选候选
- 数字键 1-9 选择候选
- Backspace 删除末位编码
- Escape 清空编码缓冲区
- Enter 提交编码原文（英文直出）
- 一级简码支持（单键 + 空格）

### 2.3 用户词典 (P1)

- 用户自定义词条（手动添加/删除）
- 词频动态调整（选中的词提升权重）
- 用户词典持久化（JSON 文件存储）
- 用户词典导入/导出

### 2.4 Z键万能键 (P1)

- Z 键作为通配符，可替代任意编码位
- 例：输入 `az` 匹配所有 `a?` 开头的编码
- 可在配置中开关此功能

### 2.5 中英文切换 (P1)

- Shift 键切换中英文模式
- 大写字母自动进入临时英文模式
- 分号键引导临时英文输入（清歌风格）

### 2.6 标点处理 (P1)

- 中文模式下自动转换中文标点
- 支持全角/半角标点切换
- 可自定义标点映射表

### 2.7 配置系统 (P1)

- TOML 格式配置文件
- 可配置项：
  - 候选词显示数量（默认5）
  - 4码自动上屏开关（默认开）
  - Z键万能键开关（默认开）
  - 词组优先/单字优先（默认词组优先）
  - 临时拼音反查开关
  - 码表文件路径

### 2.8 临时拼音反查 (P2)

- 反引号键激活临时拼音模式
- 输入拼音显示对应汉字及其五笔编码
- 辅助学习五笔编码

## 3. 技术架构

```
┌─────────────────────────────────────┐
│         应用层 (CLI / macOS IMK)      │
├─────────────────────────────────────┤
│         输入引擎 (InputEngine)        │
│  ┌──────────┐ ┌──────────────────┐  │
│  │编码缓冲区 │ │ 候选生成器/排序器  │  │
│  └──────────┘ └──────────────────┘  │
├─────────────────────────────────────┤
│         码表引擎 (DictEngine)         │
│  ┌──────┐ ┌──────┐ ┌────────────┐  │
│  │ Trie │ │HashMap│ │ 用户词典    │  │
│  └──────┘ └──────┘ └────────────┘  │
├─────────────────────────────────────┤
│         配置管理 (Config)             │
└─────────────────────────────────────┘
```
